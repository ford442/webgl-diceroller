This directory holds the Emscripten-compiled WASM artifacts:

  dice_physics.js    — Emscripten module loader (ES module, generated)
  dice_physics.wasm  — Compiled WebAssembly binary (generated)

These files are generated by running:

  npm run build:wasm        (from the repository root)
  # or
  cd src/wasm && ./build.sh

Prerequisites: Emscripten SDK must be installed and activated.
See docs/WASM_ENGINE.md for full setup instructions.

NOTE: The generated .js and .wasm files are excluded from version control
via .gitignore.  Commit only this README.
